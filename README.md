# Selenium Hybrid Automation Framework - ShoppersStack

This project demonstrates a Hybrid Test Automation Framework built using Selenium WebDriver with Java.

## Tools & Technologies
- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)
- WebDriverManager

## Framework Structure
base -> Browser setup and driver initialization
pages -> Page Object classes
testCases -> TestNG test scripts
